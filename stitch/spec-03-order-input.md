# PROFIT LAB — 주문 입력 페이지 (Benchmark Order Input)

## 디자인 기준
- `mockup-reference.html` 과 동일한 픽셀 레트로 스타일 적용
- 모든 테두리는 직각 (border-radius: 0)
- 폰트: Press Start 2P (라벨/UI), JetBrains Mono (숫자), Pretendard (본문 한글)

---

## 색상 팔레트

| 이름 | 용도 | Hex |
|------|------|-----|
| 배경 | 페이지 전체 배경 | `#0a0a1a` |
| 패널 | 섹션 배경 | `#12122a` |
| 입력 배경 | 텍스트 입력 필드 | `#0a0a1a` |
| 패널 진하게 | 주문 카드 배경 | `#1a1a4e` |
| 테두리 | 모든 테두리 기본색 | `#3355ff` |
| 시안 | 버튼/강조, 가용잔액 | `#00eeff` |
| 옐로우 | 타이틀 | `#ffe000` |
| 그린 | TP 가격, Long, 높은 confidence | `#00ff7f` |
| 레드 | SL 가격, Short, 낮은 confidence | `#ff3333` |
| 옐로우 | 중간 confidence | `#ffe000` |
| 흰색 | 기본 텍스트 | `#f0f0ff` |
| 회색 | 라벨/보조 텍스트 | `#8888aa` |

## 컴포넌트 상세

### 1. 페이지 타이틀
- 폰트: Press Start 2P, 13px
- 색상: `#ffe000` (옐로우)
- 텍스트 그림자: `2px 2px 0 #886600, 4px 4px 0 #443300`
- 내용: `★ 주문 입력`
- 서브텍스트: Pretendard 13px, `#8888aa`

### 2. 리더보드 링크 버튼 (우측)
- 폰트: Press Start 2P 7px
- 배경: 투명
- 테두리: `2px solid #3355ff`
- 색상: `#8888aa`
- 내용: `리더보드`

### 3. 섹션 패널 공통
- 배경: `#12122a`
- 테두리: `2px solid #3355ff`
- 패딩: 20px 24px
- 섹션 레이블: Press Start 2P 8px, `#00eeff` (`■ 섹션명`)

### 4. 모델 이름 입력 필드
- 너비: 300px 고정
- 배경: `#1a1a4e`
- 테두리: `2px solid #3355ff`
- 폰트: JetBrains Mono 13px, `#f0f0ff`
- 라벨: Press Start 2P 7px, `#8888aa`

자동완성 드롭다운:
- 배경: `#12122a`
- 테두리: `2px solid #3355ff`
- 항목: Pretendard 13px, `#f0f0ff`
- 호버: 배경 `#1a1a4e`
- 항목 구분선: `1px solid rgba(51, 85, 255, 0.3)`

### 5. 가용 잔액 표시 (모델명 입력 시 표시)
- 라벨: Press Start 2P 7px, `#8888aa`
- 숫자: JetBrains Mono 22px bold, `#00eeff`
- 형식: `$84.30`

### 6. 시장 분석 텍스트에어리어
- 배경: `#1a1a4e`
- 테두리: `2px solid #3355ff`
- 폰트: Pretendard 13px, line-height 1.7
- 최소 높이: 8행 (rows=8)
- 플레이스홀더 색: `#8888aa`

### 7. 주문 목록 헤더
버튼 그룹:
- `전체 삭제`: Press Start 2P 7px, 배경 투명, 테두리 `2px solid #3355ff`, `#8888aa`
- `+ 주문 추가`: Press Start 2P 7px, 배경 `rgba(0, 238, 255, 0.08)`, 테두리 `2px solid #00eeff`, `#00eeff`

### 8. 주문 카드 (ORDER 01, 02, ...)

카드 전체:
- 배경: `#1a1a4e`
- 테두리: `2px solid rgba(51, 85, 255, 0.5)`
- 패딩: 16px 18px

카드 헤더:
- 좌측: `ORDER 01` — Press Start 2P 7px, `#00eeff`
- 우측: `× 삭제` — Press Start 2P 7px, `#8888aa`

**Row 1: 핵심 주문 필드 (7열 그리드)**

| 필드 | 너비 | 내용 |
|------|------|------|
| 코인 | 90px | 드롭다운 (BTC, ETH...) |
| 방향 | 80px | LONG (초록) / SHORT (빨강) |
| 유형 | 80px | LIMIT / MARKET |
| 진입가 | 1fr | 숫자 입력, 우측 정렬, 흰색 |
| TP1 | 1fr | 숫자 입력, 우측 정렬, 초록 |
| TP2 | 1fr | 숫자 입력, 우측 정렬, 초록 (선택) |
| SL | 1fr | 숫자 입력, 우측 정렬, 빨강 |

모든 입력 필드:
- 배경: `#0a0a1a`
- 테두리: `2px solid #3355ff`
- 폰트: JetBrains Mono 13px

방향 드롭다운 색상:
- LONG: `#00ff7f`
- SHORT: `#ff3333`

**Row 2: Confidence + 근거 (2열 그리드)**

Confidence 버튼 (1~5):
- 크기: 32×32px
- 테두리 2px solid

| 레벨 | 배경 | 테두리/텍스트 |
|------|------|-------------|
| 1~2 | `rgba(255,51,51,0.15)` | `#ff3333` |
| 3 | `rgba(255,224,0,0.15)` | `#ffe000` |
| 4~5 | `rgba(0,255,127,0.15)` | `#00ff7f` |
| 비활성 | 투명 | `#3355ff` / `#8888aa` |

라벨: Press Start 2P 7px, `#8888aa`, `Confidence (3/5)`

근거 텍스트에어리어:
- 배경: `#0a0a1a`
- 테두리: `2px solid #3355ff`
- Pretendard 13px, 3행

### 9. 오류 메시지 박스 (오류 발생 시)
- 배경: `rgba(255, 51, 51, 0.08)`
- 테두리: `2px solid #ff3333`
- 텍스트: Pretendard 13px, `#ff3333`

**리더보드 버튼:**
