# PROFIT LAB — 전략 검증 페이지 (Backtest)

## 디자인 기준
- `mockup-reference.html` 과 동일한 픽셀 레트로 스타일 적용

## 컴포넌트 상세

### 1. 페이지 타이틀
- 폰트: Press Start 2P, 13px
- 색상: `#ffe000` (옐로우)
- 텍스트 그림자: `2px 2px 0 #886600, 4px 4px 0 #443300`
- 내용: `▶ 전략 검증`
- 서브텍스트: Pretendard 14px, `#8888aa` (전략 설명)

### 2. 전략 탭 버튼
- 폰트: Press Start 2P, 7px
- 패딩: 8px 14px
- 테두리: 2px solid

**비활성 상태:**
- 배경: `#12122a`
- 테두리: `#3355ff`
- 텍스트: `#8888aa`

**활성 상태:**
- 배경: `rgba(0, 238, 255, 0.12)`
- 테두리: `#00eeff`
- 텍스트: `#00eeff`

탭 목록: `RSI DIV` / `EMA TREND` / `BB SQUEEZE`

### 3. 섹션 패널
- 배경: `#12122a`
- 테두리: `2px solid #3355ff`
- 패딩: 20px 24px
- 섹션 레이블: Press Start 2P 8px, `#00eeff`
  - 형식: `■ 섹션명`

### 4. 날짜 입력 필드
- 배경: `#1a1a4e`
- 테두리: `2px solid #3355ff`
- 폰트: JetBrains Mono, 13px, `#f0f0ff`
- 라벨: Press Start 2P 7px, `#8888aa` (uppercase)

### 5. 코인 토글 버튼

**비선택:**
- 배경: `#1a1a4e`
- 테두리: `2px solid #3355ff`
- 텍스트: JetBrains Mono 11px, `#8888aa`

**선택됨:**
- 배경: `rgba(0, 238, 255, 0.15)`
- 테두리: `2px solid #00eeff`
- 텍스트: JetBrains Mono 11px, `#00eeff`

전체 선택/해제 버튼:
- 배경: 투명
- 테두리: `2px solid #3355ff`
- 폰트: Press Start 2P 7px, `#8888aa`

### 6. 전략 상세 규칙 (접힘/펼침)
- 토글 버튼: `■ 전략 상세 규칙 ▼`
- Press Start 2P 8px, `#00eeff`
- 본문: Pretendard 13px, `#f0f0ff`, line-height 1.8
- 항목 색상:
  - Long/수익 표시: `#00ff7f`
  - Short/손실 표시: `#ff3333`
  - 보조 강조: `#00eeff`

### 7. 실행 버튼

**대기 상태:**
- 배경: `rgba(0, 238, 255, 0.12)`
- 테두리: `3px solid #00eeff`
- 폰트: Press Start 2P 9px, `#00eeff`
- 텍스트 그림자: `0 0 10px #00eeff`
- 박스 그림자: `0 0 12px rgba(0, 238, 255, 0.2)`
- 내용: `▶ 백테스트 실행`

**실행 중 상태:**
- 배경: `#1a1a4e`
- 테두리: `3px solid #8888aa`
- 텍스트: `#8888aa`
- 내용: `▶▶ 실행 중...`

### 8. 진행률 바 (실행 중에만 표시)
- 트랙: 높이 12px, 배경 `#1a1a4e`, 테두리 `2px solid #3355ff`
- 채움 색상:
  - 0~30%: `#ff3333` (레드)
  - 30~70%: `#ffe000` (옐로우)
  - 70~100%: `#00ff7f` (그린)
- 우측에 퍼센트: JetBrains Mono 12px, `#00eeff`
- 좌측에 상태 메시지: Pretendard 12px, `#8888aa`
