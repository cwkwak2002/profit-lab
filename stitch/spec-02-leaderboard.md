# PROFIT LAB — AI 리더보드 페이지 (Benchmark Leaderboard)

## 디자인 기준
- `mockup-reference.html` 과 동일한 픽셀 레트로 스타일 적용

## 컴포넌트 상세

### 1. 페이지 타이틀
- 폰트: Press Start 2P, 13px
- 색상: `#ffe000` (옐로우)
- 텍스트 그림자: `2px 2px 0 #886600, 4px 4px 0 #443300`
- 내용: `★ AI 리더보드`
- 서브텍스트: Pretendard 13px, `#8888aa`

### 2. 주문 입력 버튼 (헤더 우측)
- 폰트: Press Start 2P 7px
- 배경: `rgba(0, 238, 255, 0.08)`
- 테두리: `2px solid #00eeff`
- 색상: `#00eeff`
- 내용: `+ 주문 입력`

### 3. 메트릭 카드 (4개)
그리드: 4열, 동일 너비, gap 8px

각 카드:
- 배경: `#1a1a4e`
- 테두리: `2px solid #3355ff`
- 패딩: 16px 20px

**라벨:** Press Start 2P 7px, `#8888aa`, uppercase, letter-spacing 0.08em

**값:** JetBrains Mono 22px bold

값 색상 기준:
- 수익률 양수: `#00ff7f`
- 수익률 음수: `#ff3333`
- 중립 (모델 수, 주문 수): `#00eeff`

서브텍스트 (최고 수익률 카드의 모델명): Pretendard 11px, `#8888aa`

### 4. 순위 테이블

테이블 전체:
- 외부 테두리: `2px solid #3355ff`
- 배경: `#12122a`

**테이블 헤더 행:**
- 배경: `#1a1a4e`
- 하단 테두리: `2px solid #3355ff`
- 폰트: Press Start 2P 6px, `#8888aa`, uppercase

헤더 컬럼: `#` / `모델` / `잔액` / `수익률` / `승률` / `MDD` / `Profit Factor` / `체결률` / `주문`

**테이블 데이터 행:**
- 행 구분선: `1px solid rgba(51, 85, 255, 0.3)`
- 호버: 배경 `rgba(51, 85, 255, 0.12)`

**순위 뱃지 (1~3위):**
